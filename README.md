# keyota-website – erster B2B-Entwurf

## Inhalt
- `index.html`: Startseite (Positionierung + Einstiege)
- `fuer-wen.html`: Zielgruppen
- `probleme.html`: Probleme & Lösungsansatz
- `einfuehrung.html`: Einführungsprozess
- `integrationen.html`: API/Webhooks/Sandbox
- `start.html`: Startprozess & Kontakt
- `styles.css`: gemeinsames Styling im KEYOTA-nahen Look

## Was bereits umgesetzt ist
- Vollständiger erster Wurf mit allen Kerninhalten der aktuellen B2B-Seite
- B2B-Fokus für Zielgruppen: Hausverwaltung, Bürobetreiber, Serviced Apartments
- Hero mit Video-Overlay im bestehenden KEYOTA-Stil
- Intro, Branchenblöcke, Vorteile Nutzer/Entscheider, Prozessschritte, Kontaktbereich
- Plattform-Abschnitt auf Basis realer PreProd-Module (Standorte, Bridges, Schlösser, Token, Benutzer, Protokolle)
- Integrations-Abschnitt (Open API V1/V2, Webhooks, API-Keys, Sandbox)
- Feature-Tabs (Plug & Play, Zutrittsverwaltung, Integration, Digitale Schlüssel, Sicherheit, Schließsysteme)
- Smart Relay als wichtigstes neues Produkt ergänzt
- Produktportfolio inkl. zusätzlicher Platzhalter für kommende Produkte
- Vorhandene keyota-Medien (Shopify CDN) eingebunden
- Tonalität mit Marktvergleich geschärft (CES, Schulte-Schlagbaum, WILKA, dormakaba, Interflex)
